# 乐队官网项目文档

## 1. 项目概述
本项目旨在为乐队创建一个现代、炫酷的官方网站。网站将以黑色和紫色为主色调，提供乐队信息、作品展示、演出经历和合作方式等内容，并通过响应式设计确保在不同设备上的良好用户体验。

## 2. 技术栈
- **HTML5**: 负责网页结构和内容。
- **CSS3**: 负责网页样式、布局和动画效果。
- **JavaScript**: 负责交互功能，如导航、时间轴动态加载和事件详情展示。

## 3. 文件结构
```
Walpurgis Web/
├── index.html          # 主页文件
├── css/
│   └── style.css       # 主样式文件
├── js/
│   └── script.js       # 主JavaScript文件
├── assets/
│   ├── images/         # 存放乐队Logo、专辑封面、单曲封面、演出图片等
│   └── fonts/          # 存放自定义字体（如果需要）
└── README.md           # 项目文档
```

## 4. 页面结构与内容

### 4.1 顶部导航栏 (Header)
- **Logo**: 乐队Logo，点击可返回主页。
- **导航菜单**:
    - 主页 (Home)
    - 乐队介绍 (About)
    - 作品 (Works)
    - 合作方式 (Contact)
- 菜单项点击后平滑滚动到对应页面区域。

### 4.2 主页 (Home Section)
- **英雄区域 (Hero Section)**:
    - 乐队Logo (大尺寸展示)
    - 乐队简介 (简短吸引人的文字)
    - “了解更多”按钮: 点击后跳转到“乐队介绍”分页面。
- **演出经历 (LiveActions Timeline)**:
    - 响应式滚动时间轴，样式参考提供的图片。
    - 每个时间轴事件包含年份、图片、标题和简短描述。
    - 点击时间轴事件可打开对应的事件详情弹窗或页面。
- **作品展示 (Works Section)**:
    - **单曲部分 (Singles)**: 展示乐队的单曲，包含封面、标题、发布日期等。
    - **专辑部分 (Albums)**: 展示乐队的专辑，包含封面、标题、曲目列表等。

### 4.3 乐队介绍 (About Section)
- 乐队详细介绍：历史、成员、音乐理念等。
- 乐队成员照片和简介。

### 4.4 作品 (Works Section)
- 详细的单曲和专辑列表，可包含播放链接、歌词等。

### 4.5 合作方式 (Contact Section)
- 联系信息：邮箱、社交媒体链接、合作表单等。

### 4.6 底部版权信息 (Footer)
- 版权声明。

## 5. 设计风格
- **主色调**: 黑色 (#000000) 和紫色 (例如 #8A2BE2, #9370DB)。
- **辅助色**: 少量亮色用于强调和互动元素。
- **字体**: 现代、简洁的无衬线字体，可搭配少量艺术字体用于标题或Logo。
- **视觉效果**:
    - 炫酷的背景动画或视差效果。
    - 按钮、链接等交互元素的悬停动画。
    - 时间轴的滚动和事件详情的弹出动画。
    - 作品展示的卡片式布局和悬停效果。
- **响应式设计**: 确保网站在桌面、平板和手机等不同设备上都能良好显示和操作。

## 6. 功能实现
- **平滑滚动**: 导航菜单点击后平滑滚动到目标区域。
- **时间轴交互**:
    - 滚动时动态加载内容（如果内容很多）。
    - 点击事件弹出详情。
- **作品筛选/分类**: （可选）如果作品数量多，可考虑添加筛选或分类功能。
- **响应式布局**: 使用媒体查询 (Media Queries) 实现不同屏幕尺寸的适配。
- **图片懒加载**: 优化页面加载性能。

## 7. 后续步骤
1. 创建 `index.html` 文件，搭建基本页面结构。
2. 创建 `css/style.css` 文件，定义全局样式和布局。
3. 创建 `js/script.js` 文件，实现交互功能。
4. 逐步填充内容和样式，实现各个模块。
